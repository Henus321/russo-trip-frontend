## Russo Trip

![App logo](/assets/logo.jpg)

Frontend of travel blog on Next.js. You can view posts, leave comments and make bookmarks. Authentication provided by Strapi.

This project deployed on vercel: [russo-trip.vercel.app/](https://russo-trip.vercel.app/)

Backend: [github.com/Henus321/russo-trip-backend](https://github.com/Henus321/russo-trip-backend)

### Project technology stack:

- TypeScript
- Next.js
- MobX
- Tailwind CSS
- Strapi
